<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| admin browser → autocomplete endpoint | Untrusted query text and pagination params enter server route |
| endpoint route → filter spec resolver | User-selected field must not escape declared autocomplete specs |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04 | S | autocomplete endpoint field selector | mitigate | Resolve field only from declared `FilterSpec` entries where `filter_kind == "autocomplete"`; reject others with 400 (D-01, D-13). |
| T-03-05 | T | request params (query/page/limit) | mitigate | Parse through strict validator and enforce bounds before any search call (D-13). |
| T-03-06 | D | endpoint pagination | mitigate | Enforce conservative page size and deterministic slicing to cap server workload (D-07, D-08, D-12). |
</threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser input → JS runtime | Untrusted free-text input drives request cadence |
| JS runtime → endpoint | Client controls query/page request frequency and ordering |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-07 | D | autocomplete.js request loop | mitigate | Debounce requests and enforce one latest-active fetch path to prevent keystroke flood (D-10). |
| T-03-08 | T | stale response ordering | mitigate | Use request token/version or AbortController; ignore responses not matching latest issued query (D-11). |
| T-03-09 | I | template payload exposure | mitigate | Keep initial render payload minimal; do not embed large or sensitive option payloads in HTML (D-03, D-09, D-12). |
</threat_model>
